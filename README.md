# Gym-Membership-Analytics
Overview

This project analyzes gym membership and member activity data using a complete ETL and business intelligence workflow. The dataset was sourced from Kaggle and transformed into an interactive Tableau dashboard to uncover trends in member engagement, gym utilization, and service usage.

The purpose of this project was to demonstrate practical experience with:

Data cleaning and validation
SQL-based exploratory analysis
Business intelligence reporting
Interactive dashboard development
End-to-end analytics workflows
Tools Used
Excel — Initial staging and pre-cleaning
SQL Server Management Studio (SSMS) — Data cleaning, validation, and exploratory analysis
Tableau Public — Interactive dashboards and data visualization
GitHub — Documentation and version control
Key Analysis Areas
Membership distribution
Weekly visit trends
Peak gym utilization hours
Group fitness participation
Personal training adoption
Sauna and beverage service usage
Member engagement behavior
Skills Demonstrated
ETL Workflow
SQL Querying
Data Cleaning & Validation
Exploratory Data Analysis (EDA)
Tableau Dashboard Design
Business Intelligence Reporting
Data Storytelling
