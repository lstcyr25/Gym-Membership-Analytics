# Gym Membership Analytics Dashboard

## 📊 Overview

This project analyzes gym membership and member activity data to uncover trends in gym engagement, service usage, and member behavior.

Working in a gym environment gave me interest in exploring operational and behavioral patterns through data analytics. Instead of relying on assumptions, I used SQL and Tableau to identify measurable insights related to member attendance, gym utilization, personal training adoption, and overall engagement.

---

## 🎯 Analytical Questions

* Which membership types are most common?
* What are the peak gym utilization hours?
* Which age groups visit most frequently?
* How does service usage vary across membership tiers?
* What factors contribute to higher member engagement?

---

## 📁 Dataset

* Source: Kaggle
* Dataset: Gym Membership Dataset
* Description: Member demographic, attendance, and service usage data from a gym environment

### 📌 Dataset Summary

* Total Records: 1,000 Members
* Data Types: Numerical, categorical, and time-based data
* Categories Included:

  * Membership types
  * Weekly attendance
  * Check-in/check-out times
  * Group fitness participation
  * Personal training usage
  * Sauna and beverage purchases
  * Demographic information

## 🧹 Data Cleaning (SQL - SSMS)

* Created staging tables to preserve raw data
* Validated NULL values and inconsistent records
* Standardized categorical fields
* Performed time and attendance validation checks
* Prepared dataset for visualization and analysis

---

## 📊 Tableau Dashboard

An interactive dashboard was created to visualize gym utilization, member engagement, and service usage trends.

### Dashboard Highlights

* Membership distribution analysis
* Weekly visits by age group
* Peak gym utilization heatmaps
* Personal training adoption trends
* Sauna and beverage service usage
* Group fitness participation analysis

---

## 📈 Key Insights

* Premium members demonstrated higher overall engagement and longer average gym sessions.
* Evening hours represented the highest facility utilization periods.
* Younger demographics showed higher average weekly attendance.
* Members using additional services tended to spend more time in the gym.
* Personal training demand varied significantly throughout the week.
